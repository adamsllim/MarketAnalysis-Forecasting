# Market Analysis & Forecasting

## Overview...
This project main goal is to analyze commodity markets using Python.
It focuses on crude oil prices and walks through the core skills used by real trading analysts:

## Project Goals...
- Work with time‑series market data
- Understand price behavior and market fundamentals
- Build predictive models
- Generate trading signals
- Evaluate model performance
- Present insights clearly

## Project Structure...
**1. Data Collection**
- Download WTI and Brent prices from Yahoo Finance
- Clean and store the data in CSV files

**2. Exploratory Data Analysis**
- Plot price trends
- Calculate moving averages
- Measure volatility

**3. Fundamentals**
- Pull U.S. crude oil inventory data
- Merge it with price data
- Explore how fundamentals relate to price movements

**4. Simple Price Forecasting**
- Build a moving average forecast
- Try a basic linear regression model
- Evaluate accuracy with MAE and RMSE

**5. Basic Trading Signal**
- Create a simple trend‑following signal
- Example: price above/below 30‑day moving average
- Backtest the signal on historical data

**6. Dashboard**
- A simple dashboard that displays:
- Price charts
- Forecasts
- Trading signals
- Fundamental indicators

**7. Performance Review**
- Evaluate forecast accuracy
- Review trading signal performance
- Summarize what worked and what didn’t
